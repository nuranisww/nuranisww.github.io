Welcome to my e-Portfolio. This portfolio presents my work and learning progress throughout the Mobile Programming course, including the development of LearnFlow, a study timer application. The project supports SDG 4: Quality Education by promoting organised and focused learning.

SDG 4: Quality Education

LearnFlow encourages students to manage study sessions efficiently and maintain consistent learning habits, fostering equitable and high-quality self-education infrastructure.

Laboratory Exercises

Core practical competencies built step-by-step using modern Android development principles.

Lab 1: UI Fundamentals

Exploratory building block focusing on understanding structural layouts and basic components in Jetpack Compose.

Reflection: Through this lab, I learned the fundamentals of Jetpack Compose layouts using components such as Column, Row, Box, and Modifiers. Recreating the interface helped me understand how UI elements are arranged and structured in Android development. This experience improved my confidence in designing user interfaces for future mobile applications.

Lab 2: State & Interaction

Capturing user interactivity and handling local state mutation within interactive views dynamically.

Reflection: In this lab, I learned how to create interactive user interfaces using State in Jetpack Compose through components such as TextField and Button. Implementing remember and mutableStateOf helped me understand how user input can trigger recomposition and dynamically update the UI. This experience improved my understanding of building responsive mobile applications.

Lab 3: Material Design Principles

Polishing apps structurally with cohesive styling presets, themes, typography setups, and crisp UI feedback.

Reflection: In this lab, I learned how to apply Material Design principles to create a more consistent and visually appealing user interface. Customising colors, typography, and card components helped improve the overall user experience, while animations made the interface feel more interactive. This experience strengthened my understanding of designing modern mobile applications using Jetpack Compose.

Lab 4: Advanced Navigation & ViewModels

Structuring complex application flows utilizing reliable decoupled state architectures and routing patterns.

Reflection: In this lab, I learned how to build a multi-screen application using Jetpack Compose Navigation and manage UI-related data with ViewModel. Implementing routes and preserving data across screens helped me understand how modern mobile applications maintain consistency and improve user experience. This lab strengthened my understanding of application structure and state management.

Lab 5: Local Database with Room

Integrating persistent infrastructure locally while adhering cleanly to the structured MVVM pattern.

Reflection: This lab helped me understand the integration of Room Database in Android development using the MVVM architecture. I learned how data is stored permanently through Entity, DAO, Repository, and ViewModel components. I also gained experience in verifying data persistence using Room and the Database Inspector even after the app is restarted.
